# randomquotegenerator
GUI-приложение «Random Quote Generator».

1. Создайте список предопределённых цитат (текст, автор, тема).

2. выберите случайным образом цитату по кнопке «выбрать случайную цитату», которая выбирает случайную цитату.

3. Выводит историю сгенерированных цитат в виде списка.

4. Реализована фильтрацию по теме
# Инструменты
  язык Python
  Библиотека Tkinter
  библиотека random
# Имя и фамилия автора
Новикова Наталья
# Запуск программы
python3  randomquotegenerator.py
